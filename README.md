# Hi, I'm Eoin

I’m a Senior Machine Learning Engineer with 5+ years of experience building production ML systems, scalable data pipelines, and real-time inference platforms across cloud and edge environments. My work focuses on computer vision, time series modelling, pose estimation, LLM/RAG systems, and end-to-end ML engineering.

I recently completed an MSc in Advanced Artificial Intelligence at University College Dublin, where my work focused on deep learning, applied AI systems, agentic AI, and time series classification research.

I’m particularly interested in the space between research and production: taking models from experimentation through to robust, usable systems.

## Featured Projects

### Financial Regime Classification with Classical and Quantum Machine Learning

Comparative modelling project exploring classical and quantum machine learning approaches for financial regime classification.

**Tech stack:** Python, PyTorch, NumPy, Pandas, scikit-learn, XGBoost, LightGBM, yFinance, PennyLane

**Focus areas:**
- Financial time series modelling
- Classical vs quantum model comparison
- Feature engineering and model evaluation

---

### Pose-Based Tennis Swing Classification

Developed a machine learning pipeline for classifying tennis swing videos using pose estimation features.

**Tech stack:** Python, OpenCV, PyTorch, CUDA, MediaPipe, NumPy, Pandas

**Result:** Built models capable of classifying tennis swing types with 96% accuracy.

**Focus areas:**
- Pose estimation
- Human movement recognition
- Time series classification
- Sports analytics
